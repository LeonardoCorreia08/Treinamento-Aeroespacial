# Treinamento WebAR - Manutenção Aeroespacial

## Descrição do Projeto
Aplicação de Realidade Aumentada desenvolvida para treinamento de técnicos e astronautas em procedimentos de manutenção de equipamentos espaciais.

## Setor
Educação / Indústria Aeroespacial

## Objetivo
Simular procedimentos de inspeção e manutenção em equipamentos espaciais usando WebAR, permitindo que usuários interajam com componentes virtuais em um ambiente seguro de treinamento.

## Funcionalidades
- Visualização de modelo 3D de equipamento espacial em AR
- 3 componentes interativos para verificação
- Sistema de feedback visual e textual
- Contador de progresso do treinamento
- Animações de rotação, pulso e feedback

## Interações Implementadas
1. **Toque** - Clicar/tocar nos componentes para verificação
2. **Feedback Visual** - Mudança de cor ao completar verificação
3. **Animações** - Rotação contínua, pulso e efeitos de celebração

## Tecnologias Utilizadas
- A-Frame 1.4.0
- AR.js 3.4.5
- JavaScript ES6
- HTML5 / CSS3

## Como Usar

1. Abra o link da aplicação no navegador do celular
2. Permita o acesso à câmera quando solicitado
3. Imprima ou exiba o marcador Hiro (https://github.com/AR-js-org/AR.js/blob/master/data/images/hiro.png)
4. Aponte a câmera para o marcador
5. Toque nos 3 componentes coloridos para completar a verificação

## Requisitos
- Navegador moderno com suporte a WebGL
- Câmera traseira funcional
- Conexão à internet
- Marcador Hiro impresso ou em tela
